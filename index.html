Aplicação Web vs Site

Uma aplicação web é um software acessado através de um navegador, com funcionalidades avançadas como sistemas de login, processamento de dados e interatividade. Exemplos são sistemas de e-commerce, redes sociais e ferramentas de produtividade online.

Já um site é um conjunto de páginas web interligadas, geralmente com foco em fornecer conteúdo estático ou semi-dinâmico, como textos, imagens e vídeos. Exemplos são sites institucionais, blogs e portais de notícias.

Conceito de Cliente e Servidor

No modelo cliente-servidor, o cliente é o dispositivo do usuário final (computador, smartphone, tablet) que acessa o conteúdo. O servidor é o computador remoto que armazena e fornece os dados da aplicação web ou site.

Hardwares Envolvidos

Os principais hardwares envolvidos em um ambiente web são:

  • Computadores/Servidores: processam e armazenam os dados.
  • Roteadores e switches: interligam os equipamentos em uma rede.
  • Modems e adaptadores: fazem a conexão com a internet.

Serviços Relevantes

Proxy: atua como intermediário entre o cliente e o servidor remoto, melhorando a segurança e o desempenho.

VPN (Virtual Private Network): cria uma conexão segura e criptografada entre o dispositivo do usuário e a rede.

Chrome V8: é o mecanismo de JavaScript do navegador Chrome, responsável por executar o código JavaScript.
